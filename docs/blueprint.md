# **App Name**: HealthSync Pro

## Core Features:

- Intelligent Product Matching Tool: A generative AI-powered tool that allows procurement officers to describe clinical needs and receive automated, relevant medical equipment recommendations from the catalog.
- Dynamic Multi-Category Catalog: Filterable and searchable catalog for Medical Equipment, Laboratory Gear, Consumables, and Packaging with PDF brochure integration.
- Centralized Administration Dashboard: Secure dashboard to manage product inventory, careers listings, testimonials, and company team members using Firestore and Firebase Storage.
- Integrated Quote & Lead Management: Store all sales inquiries and quote requests directly in a database with automatic timestamping and status tracking.
- Career Recruitment Portal: Allows candidates to browse listings and upload CVs/Resumes directly to Firebase Storage with form data logged in Firestore.
- Financial Solutions Calculator: Interactive tool for hospitals to estimate Rent-To-Own and Long-Term payment plans for premium medical hardware.
- Managed CMS Components: Dynamic logo carousels for clients and partner visibility, along with a QR code management section editable via the Admin interface.

## Style Guidelines:

- Primary Color: Corporate Teal (#007979) representing clinical innovation and healing.
- Secondary Color: Deep Navy Blue (#0D0B61) for high-end professionalism and trust.
- Background Color: Soft Cloud Gray (#F2F7F7), maintaining 10% saturation of the primary teal for a cohesive light-mode aesthetic.
- Accent: Cool Seafoam (#70C1B3), which sits roughly 30 degrees from the primary teal to provide highlights and contrast.
- Font Pairing: 'Poppins' for modern, authoritative headlines and 'Inter' for clear, objective body text. 'Note: currently only Google Fonts are supported.'
- Thin-line, geometric medical icons used to represent laboratory, surgical, and technical support sectors with consistent weight.
- Generous whitespace with structured grid layouts inspired by clinical laboratory efficiency, utilizing a 12-column system.
- Smooth scroll-triggered transitions and subtle micro-interactions on cards and call-to-action buttons for a premium digital experience.
